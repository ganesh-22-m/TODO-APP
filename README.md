# ✅ TODO App

A simple and responsive TODO (Task Management) application built using HTML, CSS, and JavaScript. This app allows users to manage daily tasks efficiently by adding, completing, and deleting tasks.

---

## 🚀 Features

* ➕ Add new tasks
* ✅ Mark tasks as completed
* ❌ Delete tasks
* 📝 Real-time task updates
* 💾 Data persistence using Local Storage *(if implemented)*
* 🎨 Clean and user-friendly interface

---

## 🧠 How It Works

The application uses JavaScript to:

* Capture user input
* Dynamically update the task list (DOM manipulation)
* Store tasks in browser local storage *(if implemented)*
* Track completed and pending tasks

---
